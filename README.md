VanillaGuide RestedXP
============
"Задумка на долгое время хочу переписать все шаги из RestedXP в VanillaGuide с 1 уровенья, конечно же с переводом на русский язык. 
============
![Скрин](https://github.com/Eynschteyn/VanillaGuide/blob/main/Images/VanillaGuide%20screen.png)


В аддоне представлены:

* "Пошаговые" руководства по зонам на основе работы RestedXP
* Подсказки и советы по различным квестам
* Интеграция: [MetaMapBWP](https://github.com/laytya/Metamap-vanilla)
* Интеграция: [TomTom Vanilla](https://github.com/cralor/TomTomVanilla)

Желательные Аддоны 
============
* Аддон для квестов: [pfQuest by Shagu](https://github.com/Eynschteyn/pfQuest)
* Аддон для гринда: [Shagukill by Shagu](https://github.com/Eynschteyn/ShaguKill)
* Аддон для интерфейса: [ShaguTweaks by Shagu](https://github.com/Eynschteyn/ShaguTweaks)
* Папка со шрифтами кирилицы: [Fonts - эту папку необходимо закинуть в корень игры WoW](https://github.com/Eynschteyn/Fonts)

Для работы аддона необходим шрифт с поддержкой кирилицы в противном случае кирилица не будет отображаться в аддоне.
============
